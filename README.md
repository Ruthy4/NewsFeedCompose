# NewsFeedCompose

NewsFeed is simple App which uses NewsAPI to get top headlines for Nigeria.

- Jetpack Compose
- MVVM Architecture
- coil
- Dagger Hilt
- Retrofit
- Kotlin Coroutines/Flow
- Room Database

# Setup

- Git Clone the repo
- Get the API Key from [NewsAPI](https://newsapi.org/)
- Add API_KEY = "YOUR NEWS API KEY" to local.properties 
- Build app and enjoy
